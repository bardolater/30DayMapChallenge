Crash Fatalities in Delaware (2021)

Description of the Data Set

The data set was imported as a CSV file on from this website: https://opendata.firstmap.delaware.gov/datasets/delaware-public-crash-data-2-0/ The public crash data set supplies the data points on the map and the {tigris} package supplies the lines, which are the main roads in Delaware. Some fatalities were not on main highways, so the points are on smaller roads that are viewable by zooming in. Points of crashes are in orange, and the major roads and highways are in blue.
